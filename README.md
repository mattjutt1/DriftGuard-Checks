# DriftGuard

A GitHub App for intelligent pull request analysis and automated checks.

## Quick Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Configure GitHub App**
   ```bash
   cp .env.example .env
   # Edit .env with your GitHub App credentials
   ```

3. **Start Development**
   ```bash
   npm run dev
   ```

4. **Deploy to Production**
   ```bash
   npm run build
   npm start
   ```
